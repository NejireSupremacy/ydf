## Installation

1. Install [**Node** v18 or later](https://nodejs.org).

2. Install nard:

    ```sh-session
    npm add nard
    yarn add nard
    pnpm add nard
    ```
3. Install discord.js:

    ```sh-session
    npm add discord.js
    yarn add discord.js
    pnpm add discord.js
    ```

## Starting

1. Run `nard init`.
