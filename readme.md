## Starting

1. Install [**Node** v18 or later](https://nodejs.org).

2. Install framework:

    ```sh-session
    npm add nard
    yarn add nard
    pnpm add nard
    ```
3. Install dependences:

    ```sh-session
    npm add discord.js
    yarn add discord.js
    pnpm add discord.js
    ```

4. Create a project:

    ```sh-session
    npm nard init
    pnpm nard init
    yarn nard init
    ```

5. Deploy a project:

    ```sh-session
    npm nard deploy
    pnpm nard deploy
    yarn nard deploy
    ```
