## Installation

1. Install [**Node** v18 or later](https://nodejs.org).

2. Install yotrd:

    ```sh-session
    npm add yotrd
    yarn add yotrd
    pnpm add yotrd
    ```
3. Install discord.js:

    ```sh-session
    npm add discord.js
    yarn add discord.js
    pnpm add discord.js
    ```

## Starting

1. Run `yotrd init`.

2. Run `yotrd deploy`.
