## Starting

1. Install [**Node** v18 or later](https://nodejs.org).

2. Install framework:

    ```sh-session
    npm add nard
    yarn add nard
    pnpm add nard
    ```
3. Install dependences:

    ```sh-session
    npm add discord.js
    yarn add discord.js
    pnpm add discord.js
    ```

4. Create a minimal or full project:

    ```sh-session
    npm nard init minimal
    pnpm nard init minimal
    yarn nard init minimal
    ```
