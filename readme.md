## Installation

1. Install [**Node** v18 or later](https://nodejs.org).

2. Install package:

    ```sh-session
    npm add disland.js
    pnpm disland.js
    yarn disland.js
    ```
3. Install core:

    ```sh-session
    npm add discord.js
    pnpm discord.js
    yarn discord.js
    ```

    ```sh-session
    npm add @biscuitland/core
    pnpm @biscuitland/core
    yarn @biscuitland/core
    ```

4. Run package:

    ```sh-session
    npm dld
    pnpm dld
    yarn dld
    ```
