## Installation

1. Install [**Node** v18 or later](https://nodejs.org).

2. Install core:

    ```sh-session
    npm add @voart/discord.js
    yarn add @voart/discord.js
    pnpm add @voart/discord.js
    ```
3. Install dependences:

    ```sh-session
    npm add discord.js
    yarn add discord.js
    pnpm add discord.js
    ```

4. Run command:

    ```sh-session
    npm vdcd
    pnpm vdcd
    yarn vdcd
    ```
